## 2. 全局契约测试

### 2.1 统一响应格式

- [x] 所有成功响应必须为：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

- [x] 所有业务错误响应必须仍包含 `code`、`message`、`data`。
- [x] 前端必须统一读取 `data` 下的业务 payload，不得把整个响应包当业务对象。
- [x] 登录、创建请求、状态查询、详单查询、管理员查询均需抽查一次响应解包。

### 2.2 鉴权与角色

- [x] 未登录访问 `/user/workspace`、`/user/task`、`/user/bills`、`/user/account`，必须跳转 `/login`。
- [x] 未登录访问 `/admin/overview`、`/admin/config`、`/admin/records`、`/admin/users`、`/admin/statistics`，必须跳转 `/login`。
- [x] `USER` 登录成功后跳转 `/user/workspace`。
- [x] `ADMIN` 登录成功后跳转 `/admin/overview`。
- [x] `USER` 访问任一 `/admin/*` 页面必须被拦截，重定向到/user/workspace。
- [x] `ADMIN` 访问普通用户页面必须被拦截或重定向到管理端/admin/overview。
- [x] 伪造、缺失、过期 token 调用任一需要登录的接口，应返回认证错误，不得返回业务数据。
- [x] A 用户不得查看或操作 B 用户的请求状态、详单、当前请求。

### 2.3 浏览器本地存储边界

- [x] 浏览器本地存储只允许影响登录会话信息，例如 `auth_token`、`user_role`、`user_id`、`username`。
- [x] 当前请求状态不得依赖 `localStorage.request_id`。
- [x] 账单列表不得依赖 `localStorage.request_ids`。
- [x] “服务端已有活跃请求”不得依赖 `localStorage.active_request_conflict`。
- [x] 手动在浏览器 Console 写入旧值：

```js

localStorage.setItem('request_ids', '["REQ9999"]')
localStorage.setItem('active_request_conflict', '1')
```

刷新页面后预期：

- [x] 当前请求页仍以 `/api/request/active` 为准。
- [x] 若后端没有活跃请求，页面显示“暂无进行中的请求”。
- [x] 账单页仍以数据库详单列表为准。
- [x] 工作台提交按钮不得因为本地旧值被错误禁用。

## 15. 普通调度 NORMAL

### 15.1 基础队列模型

- [ ] 等候区容量为 `WaitingAreaSize`。
- [ ] 每个桩队列容量为 `ChargingQueueLen`，且包含队首充电位。
- [ ] 快充桩只接收 `FAST` 请求。
- [ ] 慢充桩只接收 `SLOW` 请求。
- [ ] 不得出现中央等待池口径。

### 15.2 叫号规则

- [ ] 当任意同模式桩队列有空位，系统从该模式等候区取排队号最靠前车辆。
- [ ] `FAST` 等候区按 `F1 < F2 < F3`。
- [ ] `SLOW` 等候区按 `T1 < T2 < T3`。
- [ ] 慢充空位不得叫快充号。
- [ ] 快充空位不得叫慢充号。

### 15.3 最短完成时间选桩

- [ ] 构造多个同模式桩都有空位。
- [ ] 构造各桩已有不同剩余工作量。
- [ ] 新车应进入 `wait_time + self_charge_time` 最小的桩。
- [ ] 当完成时间相同，应按系统稳定规则选择固定桩，结果必须可重复。

### 15.4 队首开始充电

- [ ] 入桩队列第 1 位车辆应进入 `CHARGING`。
- [ ] 第 2 位车辆应为 `QUEUED`。
- [ ] 第 1 位完成后，第 2 位自动进入 `CHARGING`。
- [ ] 预计开始/完成时间随队列变化更新。

- [ ] 故障队列清空后，等候区叫号恢复。

- [ ] 切换故障策略为 `TIME_ORDER`。

- [ ] 标记 `FAST_01` 故障。
- [ ] 正在充电的 `F1` 生成 `FAULT_INTERRUPTED` 详单。
- [ ] 未开始充电集合应包含故障桩未开始车辆和其它同类型桩未开始车辆。
