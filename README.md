# BUPT-Charging

北京邮电大学软件工程课程项目：智能充电桩调度计费系统。

本项目面向校园充电场景，围绕“缩短等待时间、提高充电桩利用率、完成透明计费”展开设计与实现，采用：

- 前端：Vue
- 后端：Python + Flask
- 数据库：SQLite
- 通信：HTTP + JSON

## 项目目标

- 支持快充/慢充两类请求
- 基于中央等待池和动态调度机制完成排队与分配
- 支持叫号确认、充电监控、详单账单、异常处理
- 支持联合验收场景下的单次模拟与批量模拟
