## 4. 当前前端通信约定

### 4.1 Axios

当前请求层实现位于：

- [request.js](/Users/zhoujia/code/SE/ChargingPile/frontend/src/api/request.js)
- [charging.js](/Users/zhoujia/code/SE/ChargingPile/frontend/src/api/charging.js)

当前约定：

- 成功时直接返回 `response.data`
- HTTP 异常通过 Promise.reject 抛出
- 业务层自行处理 `code` 与 `message`

### 4.2 当前任务页

当前任务页实现位于：

- [Task.vue](/Users/zhoujia/code/SE/ChargingPile/frontend/src/views/user/Task.vue)

当前口径：

- 以后端 `status` 为主
- 轮询真实状态
- 消费真实的预计等待 / 开始 / 结束时间
- 消费真实详单 / 账单
